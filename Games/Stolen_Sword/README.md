# **Stolen_Sword** 

---

<br>

## **Description 📃**
- A sword used to suppress the evil force was stolen, releasing a lot of demons. 
- Find the theft and beat the demons!


## **How to play? 🕹️**
- Controls:
	- Drag to aim and release to jump and slash through enemies.
