
# Wault-Master
A TailwindCSS based Javascript game in which the user must guess the completely random vault code in order to win.


## Features

- Fully Responsive (Cross-platform)
- Written in Tailwind and Pure JS.
- IOS-Like design.
- Dark mode. (no toggle, yet.)
