# **Glitch_Buster** 

---

<br>

## **Description 📃**
- Can you fix all glitches before time runs out?
- Avoid the dangerous pointers, or kill them by jumping on their heads!

## **How to play? 🕹️**
- Controls:
	- arrow keys/WASD to move
	- hold [SPACE] to throw a breakpoint
	- [P] to pause
